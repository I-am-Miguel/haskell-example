# haskell-example

## Setup

-------------
#### Requirements
* GHCI

-------------
#### Set Up
Clone esse projeto:

~~~~bash
$ git clone https://github.com/I-am-Miguel/haskell-example.git
~~~~
Carregue os aquivos .hs
~~~~bash
cd haskell-example
$ ghci
Prelude> :l aula_01_soma_n_inteiros.hs
~~~~
Execute as funções do arquivo carregado
~~~~bash
*Main> soma(10)
55
~~~~
Sair
~~~~bash
*Main> :q
$
~~~~
